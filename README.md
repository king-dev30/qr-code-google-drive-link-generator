## 🌟 Why Use GDriveDirect?

When sharing files via Google Drive, users are normally sent to a preview page and have to click "Download" manually. For large files (over 25MB), Google also shows a mandatory virus scan warning page. 

**GDriveDirect solves this by:**
- ⚡ **1-Click Downloads:** Bypassing the preview page completely.
- 🛡️ **Bypassing Virus Scan Warnings:** Immediately downloading large files without the extra warning screen.
- 🔗 **Hiding Original Links:** Masking your original Google Drive source URL.
- 📱 **QR Code Support:** Automatically generating QR codes for mobile-friendly sharing.
- ⏱️ **Countdown Timers:** Options to add a customizable countdown before the download starts.

## 🛠️ How to Use

1. Go to your Google Drive and right-click the file you want to share.
2. Select **"Share"** and ensure general access is set to **"Anyone with the link"**.
3. Copy the link.
4. Visit [GDriveDirect.com](https://gdrivedirect.com).
5. Paste your link, choose your preferred download type (Direct, Button, or Countdown), and click **Generate**.
